# System Requirements

## Minimum Hardware Requirements

To ensure reliable operation and acceptable performance, the following minimum hardware specifications are required:

### Memory (RAM)

* Minimum: **32 GB RAM**

### Processor (CPU)

* **Intel Core i7** processor or equivalent
* Minimum **4 CPU cores**
* Minimum clock speed of **2.0 GHz**

### Storage 

* SSD sotrage is preferred over HDD
* At Least 40GB of Free Storage

### Graphics Processing Unit (GPU)

* GPU is **not required**
* A dedicated or integrated GPU is **recommended** to improve performance where applicable

## Notes

* Systems that exceed these requirements may experience better performance and responsiveness.
