# AuraNet Threat Protection Profile

This document lists  prominent web-based attack vectors that the AuraNet architecture has successfully managed to protect against.

## Successfully Mitigated Attack Vectors

1. SQL Injection (SQLi)
2. Cross-Site Scripting (XSS)
3. Cross-Site Request Forgery (CSRF)
4. Server-Side Request Forgery (SSRF)
5. Broken Authentication and Session Management
6. Insecure Direct Object References (IDOR)
7. XML External Entity (XXE) Injection
8. Remote Code Execution (RCE)
9. Path Traversal / Directory Traversal
10. Command Injection
11. Credential Stuffing
12. API Injection
13. Insecure Deserialization
14. Arbitrary File Upload
15. Cross-Site Script Inclusion (XSSI)
16. Parameter Pollution
17. DOM-based XSS
18. integer overflow
19. Malformed Packet Injection
